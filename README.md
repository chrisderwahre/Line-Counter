# Line-Counter
Tool for checking all Lines in a folder

## Usage
`Linecounter.exe "path to folder"`
Example:
`LineCounter.exe "C:/Users/Chris/TestFolder"`

Result:
```
Total Lines: 430

Total Files: 5

Folder Path: C:/Users/Chris/TestFolder

Time spend while reading: 0,0680465 seconds
```
